# ディレクトリ構成（許可リスト）

## ルート直下許可ディレクトリ

| ディレクトリ名 | 用途 |
|---|---|
| app | Next.js App Router |
| components | React コンポーネント |
| lib | ユーティリティ・共通ロジック |
| prisma | Prisma スキーマ・マイグレーション |
| public | 静的アセット |
